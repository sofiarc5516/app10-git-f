    ¿Que es HTML?

    Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código para la definición de contenido de una página web, como texto, imágenes, entre otros.

    Elementos y Atributos

    Elementos

    Las etiquetas junto con su contenido, forman lo que conocemos como un elemento HTML.
    Todos los elementos HTML se pueden dividir en dos tipos básicos:

  • Elementos en bloque: Un elemento en bloque es aquel que contiene un bloque significativo de contenido que debería mostrarse en su propia línea. Por ejemplo: párrafos, tablas, listas, etc…
  • Elementos en línea: Los elementos en línea usualmente contienen menos texto y se muestra junto a otro texto en la misma línea, cómo cuando se enfatizan palabras en una oración.

    • Atributos

    La etiqueta de apertura de un elemento puede contener atributos que proporcionan más información sobre el elemento. Los atributos deben aparecer solo en las etiquetas de apertura. Algunos elementos requieren ciertos atributos de forma obligatoria, mientras que otros son opcionales.

    Comentarios

    Los comentarios en HTML son notas que no se muestran en el navegador y que se pueden leer revisando el código fuente.

    HTML5

    HTML5 no es una reformulación de versiones previas. HTML5 incluye la redefinición de elementos, la inclusión de nuevos elementos e incluye el uso de nuevas tecnologías como video, audio y canvas.

    Formularios en HTML5

    Un formulario permite a un usuario ingresar datos. La información que se ingresa se manda a un servidor en un grupo de datos que consisten del nombre de los elementos y sus valores. Para hacer uso de un formulario, utilizaremos las etiquetas: (form)(/form). Requiere de los atributos action y method, los cuales son la URL del form-handler y el método HTTP por el cual se envían los datos, respectivamente.

  • El valor por default de action es la URL del documento actual
  • El valor por default de method es get (el otro valor válido es post)

    ¿Qué es CSS?

  • Cascading Style Sheets (CSS) ‘Hoja de estilo en cascada’ es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.
  • CSS separa la estructura de un documento de su presentación.
  • La información de estilo puede ser definida en un documento separado o en el mismo documento HTML.

    Pseudo-Clases para formularios

    Un pseudo-clases CSS consta de una clave precedida de dos puntos (:) que añadiremos al final del selector para indicar que daremos estilo a los elementos seleccionados solo cuando estos se encuentren en un estado determinado. Por ejmplo podríamos querer dar estilo a un elemento cuando éste se muestre al pasarle el puntero del ratón, o una caja de selección al estar habilitada o deshabilitada, o cuando un elemento es hijo directo de su padre el árbol DOM.

    Prefijos en CSS

    En CSS3 existen propiedades nuevas que cada navegador interpreta de forma diferente. Al no haber un estándar por el que se tengan que regir los navegadores, se deben usar los llamados “prefijos”. Los principales prefijos css: -webkit-, -moz-, -o-, -ms- se anteponen al nombre de la propiedad del estilo para que el navegador web reconozca esa determinada característica. Se aplican normalmente a estilos css3 en experimentación que aún no forman parte de una especificación o estándar.

    Propiedad Gradient

    Hay dos tipos principales de Gradientes:

  • Linear
  • Radial

    • Propiedad Transform

    Esta propiedad aplica una transformación a un elemento en 2D o 3D. Permite: rotar, escalar, mover, sesgar, etc.
